# Reference material

Books, papers and external code consulted while designing ALADYN.
**Not** distributed with the package — keep proprietary content out of
version control.

Recommended primary references:

- A. A. Shabana, *Computational Dynamics*, 3rd ed., Wiley, 2010.
- A. A. Shabana, *Dynamics of Multibody Systems*, 4th ed., Cambridge, 2013.
- E. J. Haug, *Computer Aided Kinematics and Dynamics of Mechanical Systems*,
  Vol. 1, Allyn & Bacon, 1989.
- P. E. Nikravesh, *Computer-Aided Analysis of Mechanical Systems*,
  Prentice-Hall, 1988.
- R. Featherstone, *Rigid Body Dynamics Algorithms*, Springer, 2008.
